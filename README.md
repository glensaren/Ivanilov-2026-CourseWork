Конструктор фотоальбомов
Веб-приложение на Django для создания цифровых фотоальбомов с возможностью выбора оформления. Позволяет пользователям загружать фотографии, группировать их в тематические альбомы и применять различные дизайн-шаблоны для персонализации.

Технологии и особенности
Backend часть построена на Django с использованием Django REST Framework для создания REST API. Основные сущности системы: пользователи, фотографии, альбомы, шаблоны оформления и теги. Реализована сложная фильтрация фотографий с помощью Q-объектов Django, позволяющая выполнять поиск по множеству критериев. API поддерживает все CRUD-операции и предоставляет эндпоинты для управления связями между сущностями (добавление фотографий в альбомы, применение шаблонов и т.д.).
